# E-commerce Automation Framework

## 📌 Project Overview
This project automates key user flows of an e-commerce application including login, cart, and checkout.

## 🔧 Tech Stack
- Selenium WebDriver
- Java
- TestNG
- Maven

## ✅ Test Scenarios
- Login (valid/invalid)
- Add to cart
- Checkout process
- Order confirmation

## 📊 Features
- Page Object Model (POM)
- Screenshot on failure
- Test reports

## ▶️ How to Run
1. Clone repo
2. Run testng.xml
